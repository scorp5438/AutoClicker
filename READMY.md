# 🖱️ AutoClicker — фермер ресурсов для MMO RPG

Автоматический кликер с поиском объектов по изображению.  
Добавляй ресурсы, настраивай паузы — скрипт сам будет кликать по найденным объектам на экране.

## 💻 Системные требования

### Операционная система
- Windows 10 / 11
- macOS 10.15+
- Linux (Ubuntu 20.04+, с правами sudo для клавиш)

### Обязательное ПО
- Python 3.8 или выше ([скачать](https://www.python.org/downloads/))

### Минимальные характеристики
- 2 ГБ ОЗУ
- Любой процессор от 2010 года
- Разрешение экрана: от 1280×720

### Дополнительно
- Мышь (очевидно)
- Игра, где нужно кликать по статичным объектам

> ⚠️ **Важно:** Скрипт работает в оконном режиме игры. Полноэкранный режим может работать, но не гарантирован.

## 📥 Установка

### Часть 1: Скачивание проекта

**Способ A — через git (для продвинутых):**
```bash
git clone https://github.com/yourusername/autoclicker.git
cd autoclicker
```
**Способ B — скачать ZIP:**

Нажми Code → Download ZIP
Распакуй в любую папку

==================================================
        AUTO CLICKER v1.0
==================================================

🎹 Горячие клавиши (работают из любой программы):
   F6     - Старт фарма всех ресурсов
   F7     - Остановка фарма
   F8     - Пауза
   F9     - Продолжить
   Ctrl+A - Добавить ресурс (из игры)
   Ctrl+Q - Выход

📦 Доступные ресурсы:
❌ Нет добавленных ресурсов.

Действия:
  [A] Добавить новый ресурс
  [D] Удалить ресурс
  [E] Включить/Выключить ресурс
  [P] Изменить паузу
  [S] Старт фарма (все ресурсы)
  [1-9] Старт с конкретным ресурсом
  [Q] Выход
📦 Добавление ресурса

Часть 1: Начать добавление

Нажми A

Часть 2: Ввести имя

```text
Введите имя ресурса (например: wood, stone, iron): wood
Часть 3: Захват образца
```

``text
Наведите мышь на объект и нажмите Enter...
Переключись в игру, наведи курсор на объект, нажми Enter
```

Часть 4: Пауза

```text
Пауза после клика (секунды, Enter = 2.0): 2.5
```

Часть 5: Готово!

```text
✅ Ресурс 'wood' успешно добавлен!
🎮 Запуск фарма
```
**Режим 1: Один ресурс**

Нажми цифру ресурса в меню (например, 1)

**Режим 2: Все ресурсы**

Нажми S

Во время фарма:

```text
🌲 Начинаем фармить 'wood'
   Пауза: 2.0с
   F7 - стоп | F8 - пауза | F9 - продолжить

🖱️ Клик по (850, 420)
   [Добыто: 1] Ждём 2.0с...
⌨️ Горячие клавиши

Клавиша	Действие
F6	Старт фарма всех ресурсов
F7	Остановка фарма
F8	Пауза
F9	Продолжить
Ctrl + A	Добавить ресурс (из игры)
Ctrl + Q	Выход
💡 Горячие клавиши работают даже когда игра активна — не нужно переключаться в консоль.
⚠️ Возможные проблемы
```
